ENG 

Given a sorted array of distinct integers and a target value, 
return the index if the target is found. 
If not, return the index where it would be if it were inserted in order.

You must write an algorithm with O(log n) runtime complexity.

Example 1:

Input: nums = [1,3,5,6], target = 5
Output: 2

Example 2:

Input: nums = [1,3,5,6], target = 2
Output: 1

Example 3:

Input: nums = [1,3,5,6], target = 7
Output: 4

Constraints:

1 <= nums.length <= 10^4
-10^4 <= nums[i] <= 10^4
nums contains distinct values sorted in ascending order.
-10^4 <= target <= 10^4

ESP

Dada una matriz ordenada de números enteros distintos y un valor objetivo,
devuelve el índice si se encuentra el objetivo.
De lo contrario, devuelva el índice donde estaría si se insertara en orden.

Debe escribir un algoritmo con complejidad de tiempo de ejecución O (log n).

Ejemplo 1:

Entrada: números = [1,3,5,6], objetivo = 5
Salida: 2

Ejemplo 2:

Entrada: números = [1,3,5,6], objetivo = 2
Salida: 1

Ejemplo 3:

Entrada: números = [1,3,5,6], objetivo = 7
Salida: 4

Restricciones:

1 <= números.longitud <= 10^4
-10^4 <= números[i] <= 10^4
nums contiene valores distintos ordenados en orden ascendente.
-10^4 <= objetivo <= 10^4
